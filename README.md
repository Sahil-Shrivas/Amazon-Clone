#  Amazon-Clone

A sleek, responsive frontend clone of Amazon's homepage built with pure **HTML**, **CSS**, and **JavaScript**. This project demonstrates layout skills, UI design, and interactive elements, providing a clean interface similar to Amazon’s product showcase experience.

---

##  What’s Inside

- **Home page layout** featuring hero banner and product sections  
- **Navigation bar** and category banners  
- **Grid-based product display** using CSS for layout and styling  
- **Basic interactivity**, like hover effects, built using vanilla JavaScript

---

##  Tech Stack

- **HTML5** — Semantic markup and page structure  
- **CSS3** — Flexbox or Grid for layout, responsive design, and styling  
- **JavaScript (Vanilla)** — Frontend interaction and dynamic behavior
